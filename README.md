# password-generator
About Project
This is a simple Password Generator web app made using Python, Flask, HTML and CSS.
The application generates random strong passwords instantly.
I built this project while learning Flask and backend development.
This project helped me understand how Python connects with HTML pages using Flask.

Features
____________________________________________________________________________
Generate random passwords
User can select password length
Simple and clean interface
Fast password generation
Beginner friendly project
